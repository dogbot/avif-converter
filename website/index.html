Installation

1. Install libavif

Using Homebrew:

brew install libavif

2. Install AVIF Converter

  1. Download and open AVIF-Converter.dmg
  2. Double-click install.sh to install
  3. Restart Finder if needed: killall Finder

Usage

  1. Right-click any image file in Finder
  2. Go to Quick Actions (or Services)
  3. Select Convert to AVIF or Convert from AVIF

The converted file appears in the same directory with the new extension.

Supported Formats

  • Convert TO AVIF: JPG, PNG, WebP, TIFF, BMP, and more
  • Convert FROM AVIF: AVIF to JPG
